# Zombie Escort TD — グレーボックスビルド

## 概要
1人の護衛（お父さん）を安全に右端まで護衛するタワーディフェンス。
現在はグレーボックス（単色図形）で全システムを検証中。

## ディレクトリ構成

```
zombie-escort-td/
├── index.html          # ゲーム本体
├── editor.html         # マップエディタ（Phaser不使用）
├── stages/
│   └── stage_01.json   # ステージデータ（エディタのJSON出力先）
├── assets/
│   ├── sprites/        # ASSET SWAP POINT: スプライト差し替えポイント
│   └── audio/          # ASSET SWAP POINT: 音声ファイル差し替えポイント
└── js/
    ├── config.js       # 定数・グローバルユーティリティ
    ├── pathfinder.js   # A* 経路探索（4方向・遮蔽物のみブロック）
    ├── audio.js        # Web Audio合成（実音声ファイル不要）
    ├── Escort.js       # 護衛エンティティ
    ├── Zombie.js       # ゾンビエンティティ
    ├── Tower.js        # タワー + 弾丸
    ├── WaveManager.js  # ウェーブ管理・スポーン制御
    ├── BootScene.js    # ステージJSON読み込み
    ├── GameScene.js    # メインゲームシーン（全システム統合）
    └── main.js         # Phaser初期化
```

## システム仕様

### 経路探索
- **A* 4方向**（斜め移動なし）
- ブロック対象：`obstacles` 配列のセルのみ
- **タワーは通過可能**（パスをブロックしない）

### カメラ
- ドラッグでパン、ホイール / ピンチでズーム（0.5〜1.25倍）
- `⌂ 護衛` ボタンで護衛へカメラ復帰
- 無操作 3秒後に自動復帰
- キーボード `H` で即時復帰

### タイムコントロール
- HUD右上のタイム表示をクリック or スペースキーでサイクル
- ⏸ 停止 → 🐢 スロー(0.25x) → ▶ 通常(1.0x)
- デバッグパネル（`⚙` or `D` キー）でも切替可能

### タワー配置
- 下部パネルでタイプを選択 → マップをタップで配置
- 遮蔽物セルには置けない（タワーがあるセルも不可）
- タワーをタップして選択 → 右クリックで売却

### 画面外インジケータ
- 画面外のゾンビ：緑の矢印でエッジに表示
- 画面外の護衛：赤い矢印

### キーボードショートカット
| キー | 機能 |
|------|------|
| Space | タイムモード切替 |
| H | 護衛へカメラ復帰 |
| G | グリッド表示 ON/OFF |
| P | デバッグパス表示 |
| D | デバッグパネル開閉 |
| Escape | タワー選択解除 |

## マップエディタ（editor.html）
- ブラウザで直接開いて使用（サーバー不要）
- 遮蔽物・スタート・ゴール・スポーンを配置
- JSON生成 → `stages/` フォルダに保存してゲームで読込

## デプロイ（Netlify）
1. GitHubにプッシュ
2. Netlifyで `zombie-escort-td` リポジトリを接続
3. Publish directory: `.`（ビルドコマンド不要）

## アセット差し替え手順
グレーボックス → 実素材への移行：
1. `assets/sprites/escort/` にスプライト画像を配置
2. `js/Escort.js` の `draw()` メソッドをスプライト描画に書き換え
3. `assets/audio/` に音声ファイルを配置
4. `js/audio.js` の `AudioSynth` クラスで `loadFile()` 利用に差し替え
